# Ruby-Enumerable

In this project we learned how to use a module inside a class. For this we created a class MyList and a module MyEnumerable. The module MyEnumerable  implements a subset of the functionality of [Enumerable](https://ruby-doc.org/core-3.0.0/Enumerable.html).
